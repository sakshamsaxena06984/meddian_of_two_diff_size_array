To run this project 
Only Compile and Run real_main.cpp file
Compilation of any other header file leads to unnecessary extension of space.
Both Username and Password for Admin portal is "Admin"(case--senstive).
Note:Keep all 3 files in same place.
Note:This project is made on LINUX terminal and command used is "clear" to clear the screen.
     If you are running it on WINDOWS terminal then replace all the system("clear") to system("cls").
